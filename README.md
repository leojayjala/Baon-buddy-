# Baon Budget Tracker

Student budgeting app built with Expo (React Native) and Firebase Authentication.

## Team Members

- Leo Jay L. Jala
- Jake Del Campo
- Viandre Paglinawan

## Run Locally (Expo)

1. `cd mobile`
2. `npm install`
3. `npm start`

## Firebase Setup

1. In Firebase Console, create/select your project and add a **Web app**.
2. Copy the `firebaseConfig` values into `mobile/firebase/config.js`.
3. Enable **Authentication → Sign-in method → Email/Password**.
